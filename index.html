فروشگاه‌ساز فارسی و متن‌باز برای کسب‌وکارهای آنلاین

مشاهده در GitHub

درباره پروژه

Persian SmartStore نسخه بومی‌شده و فارسی‌شده‌ی فروشگاه‌ساز متن‌باز Smartstore است که بر پایه ASP.NET Core 9 توسعه یافته و برای بازار ایران و کاربران فارسی‌زبان بهینه شده است.

این نسخه با پشتیبانی کامل از زبان فارسی، راست‌چین (RTL)، تقویم شمسی و فونت فارسی منتشر شده و با جدیدترین نسخه‌های Smartstore سازگار است.

در حال حاضر نسخه‌های زیر از اسمارت استور فارسی‌سازی شده و آماده استفاده هستند:

  • نسخه 5.1.0
  • نسخه 6.0.0
  • نسخه 6.1.0
  • نسخه 6.2.0

برای شروع، کافی است نسخه موردنظر را از بخش برنچ‌ها (Branches) در GitHub انتخاب و دریافت کنید.

امکانات فروشگاه‌ساز اسمارت استور فارسی

  • پشتیبانی کامل از زبان فارسی و RTL
  • تقویم شمسی در مدیریت و فرم‌های ورودی
  • پنل مدیریتی فارسی‌شده و قابل استفاده آسان
  • ماژولار و قابل توسعه برای برنامه‌نویسان
  • مدیریت محصولات حرفه‌ای (فیزیکی، دیجیتال، ترکیبی)
  • قابلیت چندفروشگاهی و چندزبانه
  • درگاه‌های پرداخت متنوع قابل اتصال
  • پشتیبانی از حمل‌ونقل، مالیات، انبار و صورتحساب
  • ویرایشگر صفحات (Page Builder) و CMS داخلی
  • سئوی قدرتمند با URLهای ساختار‌یافته و قابلیت ویرایش متاها
  • سازگار با GDPR و سیاست‌های امنیت داده
  • موتور قالب Liquid برای ایمیل و پیامک
  • قابل نصب روی ویندوز، لینوکس و مک
  • پشتیبانی از دیتابیس‌های SQL Server، MySQL، PostgreSQL و SQLite
  • قابل استقرار در محیط‌های ابری و با Docker

چرا از Persian SmartStore استفاده کنیم؟

این پروژه با هدف بومی‌سازی پلتفرم اسمارت استور و فراهم کردن یک گزینه قدرتمند، رایگان و مدرن برای فروشگاه‌های فارسی‌زبان توسعه یافته است. شما می‌توانید بدون پرداخت هزینه مجوز، یک فروشگاه کامل با امکانات پیشرفته داشته باشید و مطابق نیازهای بومی آن را شخصی‌سازی کنید.

شروع کنید

برای راه‌اندازی فروشگاه خود با Persian SmartStore:

  1. وارد مخزن GitHub پروژه شوید.
  2. بر اساس نسخه موردنظر خود، برنچ مربوطه را انتخاب و کد را دریافت کنید.
  3. با استفاده از دستورالعمل موجود در README، پروژه را اجرا و پیکربندی کنید.
